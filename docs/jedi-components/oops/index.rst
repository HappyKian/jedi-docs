####
OOPS
####

OOPS is the **Object Oriented Prediction System**.

It is the top level of JEDI that orchestrates the configuration and execution of applications.  This is where mose DA applications are implemented.  OOPs also includes the Lorentz 95 and Quasi-geostrophic toy models that can used to run many of the DA applications.
